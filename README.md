### Overview 
Ultra OS is a web-based operating system (browser-based desktop simulation) created to provide an interactive desktop experience directly in the browser. It is a project that combines front-end technologies for creating windows, icons and basic applications.

### 🛠 Technologies & Tools
The project was implemented using the following technologies:

* **HTML5:** For the structure of the page and desktop elements.

* **CSS3:** For styling, layout, effects (blur/glassmorphism) and element placement.

* **JavaScript**: For system functionality, such as drag-and-drop of windows, opening/closing applications and basic mathematical operations of the calculator.

### 🚀 Features
* **Draggable Windows:** All windows can be moved around the desktop.

* **Z-Index Management:** Dynamic management of window stacking (the active window always comes first).

### Applications:

* **Ultra Notes:** Integrated text editor for notes.

* **Ultra Calculator:** Functional calculator.

* **Contact Me:** Contact information and links.

* **My Projects:** View the creator's projects.

* **System Bar:** Taskbar with real-time time and battery status.

### Installation
1. Clone the repository


git clone https://github.com/travletothefurureprogramming/UltraOS

cd UltraOS

2. Open the project
Start a local server:


python3 -m http.server 8000

Then visit http://localhost:8000 in your browser.
